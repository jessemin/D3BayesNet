Project: D3 BayesNet

Interactive Bayesian Network Visualization with D3 for Non-domain experts

This is the final project of CS448B: Data Visualization, a Stanford CS course on data visualization techniques (Fall 2017). With flood of data these days, Bayesian network is widely used to deduce causal relationships from massive data. Because Bayesian network is a complete model for variables and their relationships, people use it to answer probabilistic queries and make predictions. There are three major reasons why Bayesian network is powerful:

  • Bayesian network is graphical models.
  • Bayesian network captures causal relationship between factors.
  • Bayesian network handles uncertainty.
Most importantly, Bayesian network can be improved by combining domain knowledge with data.

While there is various software designed to help people create and design a Bayesian network, only few software helps people explore possible networks by incorporating domain expertise. Even worse, up to my knowledge, there is no D3 application that facilitates interactive Bayesian network design (most of them are either standalone software written in Java or MVC). To address this problem, I built an interactive Bayesian network application in D3 for my final project.

Problem

People without domain knowledge should also be able to construct a reasonable Bayesian network.

Solution

Implement a D3 application that helps non-domain experts construct and visualize Bayesian networks.

Key Features

  1. Visualize each node as pie chart of event space.

  2. Automatically populates probability distribution table of a single variable.

  3. Suggest next action that will likely improve the network.

  4. Show conditional probability distribution of multiple variables.

  5. Interactive and lightweight web-based.
